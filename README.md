# Osu-Mania-7K-to-6K-Chart-Converter

把 osu!mania 的 7k 谱转成 6k 谱

## 使用方法

控制台输入: python main.py

## 说明
 ① 关键在于怎么修改第 4 轨道, 这个工具只是我的思路, 有其他的想法直接修改 process_hitobjects_block_tools.py 中的 trans_4th_track

 ④ 为什么做这个工具? 因为我 4k 手速不够, 7k 大拇指摁不动, 6k 的谱面又比较少. 如果想把 6k 作为 7k 的练习过渡, 这个工具也可以拿来用
